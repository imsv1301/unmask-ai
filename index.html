How to use UnMask.AI

Step 1: Get an Anthropic API key

  1. Go to console.anthropic.com
  2. Sign up free with Google or email
  3. Click API Keys in the sidebar
  4. Click Create Key and copy it
  5. Go to Billing and add $5 credit

Step 2: Paste your API key above and click Save

Step 3: Paste text and click Detect AI or Humanize

How does the 3-pass pipeline work?

Pass 1: Deep restructuring — not paraphrasing. Changes argument flow, injects perplexity, forces burstiness variation, eliminates 30+ banned AI words.

Pass 2: Silent audit — a separate AI call finds every remaining tell that any detector would flag.

Pass 3: Final revision — fixes all audit findings and produces the final undetectable text.

Which detectors does it bypass?

GPTZero, Turnitin (incl. AIR-1 + anti-humanizer 2025), Originality.ai, QuillBot, ZeroGPT, Copyleaks, iThenticate, Compilatio, PlagScan, and any detector using perplexity + burstiness analysis.
